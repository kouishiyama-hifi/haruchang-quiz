🍽️ はるちゃん食堂クイズ【2周年記念】
はるちゃん食堂の2周年を記念した公式クイズゲームです！ 放送を見てきた常連さんなら何問解けるかな？あなたの「常連度」を試してみよう！

🎮 遊び方
難易度を選んでスタート！
制限時間 20秒以内 に4択から正解を選ぼう
早く答えるほど高得点になるよ
全問クリアで結果＆称号が発表されます
🏆 難易度
難易度	コース名	問題数	倍率
🟢 初級	お通しコース	7問	×1
🟡 中級	常連コース	7問	×1.5
🔴 上級	超常連コース	6問	×2
難易度が上がるほどスコア倍率もアップ！上級はかなりマニアックな問題が出るよ👀

🎁 全問正解でご褒美あり！
どの難易度でも全問正解すると、はるちゃんからのスペシャル特典動画が見られる QRコードが出現！ ぜひ全難易度でパーフェクトを目指してみてね🌟

👑 称号システム

〜39%	🍚 一見さん
